# Calculator Using JavaScript

This is a simple **Calculator** project built using **HTML, CSS, and JavaScript**.

## About Project
This calculator can perform basic arithmetic operations.
It is created for practice and learning JavaScript fundamentals.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear / Reset functionality
- User-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Open the project folder
2. Open `index.html` in any web browser
3. Use the calculator to perform calculations

## Purpose
This project is made to understand basic JavaScript logic and DOM manipulation.
